=== WP-Bookmark-Bloginy ===
Contributors: inalgnu
Tags: post

== Description ==

This plugin add a bloginy button to vote for your post.


== Installation ==

1. Upload 'wp-bookmark-bloginy' to the '/wp-content/plugins/' directory.
2. Activate 'wp-bookmark-bloginy' through the 'Plugins' menu in WordPress.
3. Choose your display mode, in the admin options.
